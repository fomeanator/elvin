# Changelog

## [0.9.0] — 2026-07-12

- Version-lockstep release with `com.lvn.engine` 0.9.0 (install every
  package from the same tag).

## [0.8.0] — 2026-07-12

- Extracted from `com.lvn.engine` (`Runtime/UI/Addressables/`) into a
  standalone optional package. Same assembly name
  (`Lvn.Engine.Addressables`), same GUIDs, same behaviour — projects upgrade
  by just adding this package next to com.unity.addressables.
